## 修改文件

本仓库为 `ONLYOFFICE/build_tools` 的 fork，主要修改如下：

### 编译配置修改

#### 1. `tools/linux/automate.py`

**修改内容**：
- QT 下载地址改为国内镜像（USTC 镜像）
- 关闭源码自动更新（`--update 0`）

**修改位置**：
```python

base.download("https://mirrors.ustc.edu.cn/qtproject/...", "./qt_source_5.9.9.tar.xz")

# 第 109 行：关闭源码更新
"--update", "0"
```

**修改内容**：
- 连接数从默认 20 改为 2000

**修改位置**：

**修改内容**：
- 开启 advancedApi

**修改位置**：
```javascript

```

#### 4. `sdkjs/common/plugins.js`

**修改内容**：

## 配置差异说明

### 完整版 vs API 精简版

#### 移除的组件（API 版）

| 组件 | 完整版 | API 版 | 说明 |
|------|--------|--------|------|
| Redis | ✅ | ❌ | 内存存储替代 |
| RabbitMQ | ✅ | ❌ | 同步转换替代 |
| Metrics | ✅ | ❌ | 监控服务 |
| AdminPanel | ✅ | ❌ | Web 管理界面 |
| MSSQL 客户端 | ✅ | ❌ | - |
| MySQL 客户端 | ✅ | ❌ | - |
| Certbot | ✅ | ❌ | SSL 在外部处理 |

| 服务 | 完整版 | API 版 |
|------|--------|--------|
| nginx | ✅ | ✅ |
| postgresql | ✅ | ✅ |

| rabbitmq | ✅ | ❌ |
| ds-docservice | ✅ | ✅ |

## 源码修改脚本

`scripts/patch_source.sh` 会执行以下修改：

```bash
#!/bin/bash
# 1. 修改连接数
sed -i 's/LICENSE_CONNECTIONS = [0-9]*/LICENSE_CONNECTIONS = 2000/' \
    server/Common/sources/contants.js

# 2. 开启 advancedApi
sed -i 's/advancedApi: false/advancedApi: true/' \
    server/Common/sources/license.js

# 3. 修改 QT 地址
sed -i 's|https://...|https://mirrors.ustc.edu.cn...|' \
    tools/linux/automate.py

# 4. 添加 connector.api 方法
sed -i '/case "externalConnectorMessage"/,/break;/ {
    /break;/ a\
    case "historyTurnOff":\
        AscCommon.History.TurnOff();\
        break;\
    case "historyTurnOn":\
        AscCommon.History.TurnOn();\
        break;
}' sdkjs/common/plugins.js

# 5. 关闭源码更新
sed -i 's/"--update", "1"/"--update", "0"/' \
    tools/linux/automate.py
```

## 镜像配置说明

### 智能镜像切换流程

```
1. 测试 GitHub 下载速度
   ↓
2. 速度 < 500 KB/s?
   ↓ 是
3. 启用国内镜像
   - Qt: 中科大镜像
   - ICU: Gitee 镜像
   - Git: 全局配置 Gitee
   ↓ 否
4. 使用官方仓库
```

### 使用方式

```bash
# 方式一：自动检测（推荐）
./build_and_package_smart.sh

# 方式二：强制使用镜像
USE_MIRROR=true ./build_and_package_smart.sh

# 方式三：强制使用官方
USE_MIRROR=false ./build_and_package_smart.sh

# 方式四：指定 Qt 镜像源
QT_MIRROR=ustc ./build_and_package_smart.sh
```

### Git 全局配置

当启用镜像时，会执行以下 Git 配置：

```bash
git config --global url."https://gitee.com/".insteadOf "https://github.com/"
git config --global url."https://gitee.com/toarujianshang/".insteadOf "https://github.com/ONLYOFFICE/"
```
